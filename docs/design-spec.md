# 设计规范 — ChessBoardGame

## 设计理念

**"魔幻像素"风格** — 结合复古像素美学与卡牌魔幻元素。UI 应具有神秘感和仪式感。

### 卡牌交互设计

#### 橙卡出牌流程
1. **PAUSE_BEFORE** (0.2s) — 释放位置停顿
2. **MOVE_TO_200** — 指数缓动至 (300, 540)
3. **PAUSE_AFTER** (0.2s) — 卡在 Reader 入口停顿
4. **SHATTER** (0.8s) — 边上升边碎片销毁（(300,540)→(300,210)）
5. 卡牌移除 → 中央展示

#### 紫卡传送流程
1. **MOVE_TO_POS** — 指数缓动至 Portal 检测区 (300, 1060)
2. **PAUSE_BEFORE** (0.2s) — 停顿
3. **PAUSE_AFTER** (0.5s) — 再停顿
4. **MOVE_TO_PORTAL** (0.35s) — 下滑进入 Portal (1230)
5. 卡牌传送至敌方手牌（transferred=true）

#### 中央展示动画
1. **PAUSE** (0.3s) — 空白停顿
2. **APPEAR** (1.2s) — 碎片聚合，自上往下扫描
3. **DISPLAY** (1.2s) — 完整展示
4. **FADE_OUT** (0.4s) — 原色→极亮 + 透明

### 碎片动画效果

#### 棋子碎片
- **密度**：12×12=144 碎片
- **销毁**：自下往上擦除线→碎片坠落+漂移+旋转+smoothstep 淡出
- **转化**：敌方粉碎 + 我方碎片聚合（从下方飞入）
- **物理**：重力 800px/s²，水平漂移 ±80px/s，旋转 ±360°/s，淡出 0.85s smoothstep
- **衰减**：`decayFragments` 缓冲区，动画结束后继续播放至自然消失

#### 卡牌碎片
- **密度**：36×36=1296 碎片
- **销毁**：边上升边粉碎，擦除线自下往上
- **出生**：碎片从散落位置飞入聚合，自上往下扫描，1.5s（1.0s 聚合+0.5s 衰减）
- **展示**：中央展示 APPEAR 改用碎片聚合
- **退牌/诅咒移除**：2.0s 碎片销毁，衰减缓冲区

#### 碎片亮度
- BlendAdd 白光叠层，飞行中渐亮，到达后缓暗

### 悬停预览
- 疫病选子：virus.png 覆盖可感染棋子，80% 不透明度
- 笼络销毁：skull.png 覆盖己方棋子
- 笼络转化：ring.png 覆盖敌方棋子
- 统一 0.15s 淡入淡出

## 音效设计

### 背景音乐
| 曲目 | 场景 | 音量 |
|------|------|------|
| menu.wav | 主菜单/配置/设置/帮助/历史 | 100 |
| config.wav | 对战配置 | 100 |
| battle_01.wav | 对局默认 | 40 |
| battle_02.wav | CardReader/Portal 触发，连子危机延长 | 40 |
| win.wav | 胜利 | 100 |
| fail.wav | 失败 | 100 |

### 战斗音乐状态机
```
NORMAL → SWITCHING_TO_02 (CardReader/Portal 触发)
→ BATTLE_02 (30s 基础，四连+30s 叠加)
→ SWITCHING_TO_01 → NORMAL
```
- 淡出 1s→stop→淡入 1s，双音轨永不重叠
- 紫卡 Portal 传送同样触发切歌

## 字体规范
- 系统 UI：`font` — 微软雅黑/黑体
- 卡牌文字：`cardFont` — 同字体，卡牌渲染专用
- 中文 UI 使用 `std::wstring`
- 卡牌名：28px，白色
- 卡牌描述：22px，浅灰，逐行拆分，行距 26px
- 主菜单按钮：48px
- 展示卡牌：字体按 scale 比例缩放（NAME 59px / DESC 46px at 0.76 scale）
