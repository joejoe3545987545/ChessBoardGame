# 执行步骤 — ChessBoardGame

## 当前开发阶段：2K 适配完成 + 卡牌系统打磨

---

## 已完成里程碑

### ✅ 基础棋盘系统
- 15×15 棋盘渲染（2K：STEP=72，棋子 68.4px）
- 五子/六子连线胜利判定 + 高亮（每方独立 winCondition）
- 三连/四连模式检测 + 自动抽卡（含方向去重标记）
- 棋子销毁/转化动画（smoothstep + opacity）
- 落子历史记录与复盘回放

### ✅ 行动点框架
- 三模式判定（双能/仅下棋/仅出牌）
- 智能消耗算法（优先专能点，保留双能点）
- 总体结算中心（AP 归零切回合，未归零继续）
- 倒计时锁定（动画期间冻结）

### ✅ 卡牌系统（4 张）
- **连击**（FORCE_DROP）：+2 落子点
- **隐忍**（CHANGE_WIN_RULE）：敌方→六子连星
- **笼络**（CONVERT_PIECE）：销毁己方+转化敌方一子
- **破釜沉舟**（SACRIFICE_HAND）：手牌回牌库+批量销毁敌方
- 出牌流程：拖到 CardReader → 湮灭 → 中央展示 → 效果触发
- 展示动画：PAUSE→APPEAR→DISPLAY→FADE_OUT
- 退牌动画：裁剪 100%→0% + 白光（1s）

### ✅ 双卡槽系统
- 槽 1 (2260, 320)、槽 2 (2260, 960)，独立堆叠（+6px 偏移）
- 双槽独立拾取/存放，自动置顶

### ✅ 渲染系统
- 统一 `activeIdx` 管理活跃卡
- 出生动画：连续插值 + 白光消退
- 退牌动画：反方向裁剪 + 亮色
- 湮灭闪烁修复：单帧跳过标记 + 抑制出生动画

### ✅ 音乐系统
- 战斗音乐串行过渡（淡出 1s→stop→淡入 1s，双音轨永不重叠）
- CardReader / 四子连星触发切歌
- 危机叠加延长（无上限）
- 返回菜单/重开正确停止所有音轨

### ✅ 2K 全面适配（2560×1440）
- Chessboard：STEP=72，START_X/Y 用 2560/1440，棋子 68.4px
- GameEngine：全部坐标 ×2，读卡器/卡槽/按钮/字体/碰撞盒
- 主菜单 UI：按钮 320×67，间距 107

### ✅ 主菜单视觉
- 灰底 + white_HR 左下 + black_HR 右上
- 30s 抛物线往返动画 + 3°/s 旋转

### ✅ 三子连星抽卡去重
- `usedMask[15][15]` 方向位掩码，3→4 扩连不重复抽卡

### ✅ 棋子动画
- 销毁：clip 1→0 + alpha 255→0
- 转化：fromColor clip→0 + toColor alpha 0→255
- smoothstep 缓动

### ✅ 笼络/破釜沉舟流程修复
- 笼络动画完成前禁止点敌方棋子
- 破釜沉舟展示后不卡退/不闪退
- 批量销毁完成恢复定时器
