### ✅ 碎片动画系统
- **棋子碎片**：12×12=144 碎片预缓存，销毁时自下往上粉碎坠落，转化时敌方粉碎+我方聚合
- **卡牌碎片**：36×36=1296 碎片预缓存，读卡器销毁/破釜沉舟退牌/诅咒移除均用碎片效果
- **卡牌出生**：1.5s 碎片聚合（自上往下扫描，碎片飞入拼成卡牌）
- **展示动画**：APPEAR 改用碎片聚合
- **碎片衰减**：动画结束后继续播放至自然淡出（`decayFragments` / `cardDecayFrags`）
- **亮度效果**：BlendAdd 白光叠层，飞行中渐亮
- **悬停预览**：疫病 virus / 笼络销毁 skull / 笼络转化 ring，0.15s 淡入淡出
- **文字行为**：生成 0.1s 淡入 / 销毁 0.1s 淡出，逐行与扫描线同步
